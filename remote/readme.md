# turtle remote control program

listen.lua runs on the turtle and control.lua runs on the pocket computer

keys:

| key | action       |
| --- | ------------ |
| w   | go forward   |
| s   | go back      |
| a   | turn left    |
| d   | turn right   |
| q   | go up        |
| z   | go down      |
| e   | mine forward |
| r   | mine up      |
| f   | mine down    |
| g   | exit         |
